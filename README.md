# Progetto Esame Big Data

Repository realizzata per l'esame di Big Data.
**Autore:** Bruno Zollino

Il progetto contiene tre esempi di database:

1. **Database relazionale SQL** - Agenzia viaggi
2. **Database NoSQL a grafo** - Neo4j - mobilità urbana di Tokyo
3. **Database NoSQL documentale** - Elasticsearch - Recensioni Steam 

## Struttura della repository

- `database_relazionale_agenzia_viaggi/`: script SQL per la gestione del database relazionale di un'agenzia viaggi.
- `database_grafo_neo4j_persona5/`: database a grafo basato sulla mobilità e i collegamenti della Tokyo di Persona 5.
- `database_documentale_elasticsearch/`: ambiente Docker e script per l'analisi delle recensioni Steam di Clair Obscur: Expedition 33 tramite Elasticsearch.
